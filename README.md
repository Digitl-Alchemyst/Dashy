<h3>
<ul><b>How to use</b></ul>
</h3>

## **Project Dependencies**

    Node.js 18.16.0
    npm/npx 9.5.1

## Packages

    MUI 5.11.2
    MUI X-Data-Grid 5.17.2
    Emotion 11.10.6
    Formik 2.2.9
    React Pro Sidebar 0.7.1
    Full Calendar 5.11.2
    Nivo 0.80.0

## Install Node Modules

    npm install

## Run Application

    npm start
